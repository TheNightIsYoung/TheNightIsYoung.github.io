Eclipse 2020 Shortcut Key

 2017-10-11 2022-03-14Java4 minutes read (About 631 words)

Eclipse 2020 版本常用快捷键。

注释快捷键

1 –> ctrl+shift+/

使用 /* */（块注释）自动注释掉选择的代码块。

问题：注释后多行语句乱码（合并为一行），取消注释无法恢复。

Solution：Window –> Perference –> Java –> Code Style –> Formatter –> 取名（my_add_annotation）–> Edit –> Comments（左侧）–> 勾掉：Enable block comment formatting –> Apply –> OK –> Apply and Close;

2 –> ctrl+shift+\

取消 /* */（块注释）注释。

3 –> ctrl+/

使用 //（行注释）自动 注释或取消注释 选择的多行代码。

4 –> ctrl+shift+j

快速为文档，类，方法等添加模板注释。

注释模板修改：

Window –> Preferences –> Java –> Code Style –> Code Templates –> Comments –> Types –> Edit；

编辑快捷键

1 –> ctrl+d

删除当前光标所在行。

2 –> ctrl+shift+enter

在当前行上插入一行。

3 –> shift+enter

在当前行下插入一行。

4 –> alt+/

自动补全代码或者提示代码。

问题：Eclipse 编写代码不能自动补全，而需要不停的按 ALT+/ 或者检测到 . 才会进行补全。

Solution：Window –> Perference –> Java –> Editor –> Content Assist –> Auto activation triggers for Java（右侧，只有一个 . 存在） –> 改为：**.abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ** –> Apply –> OK;

这样，就可以通过代码补全，自动生成 main，getter，setter，toString 等方法了。

5 –> ctrl+shift+o

自动引入包和删除无用包。

6 –> ctrl+shift+f

格式化代码。

问题：使用搜狗输入法/微软输入法，会和 “繁/简体切换” 快捷键冲突。

Solution：关闭输入法 “繁/简体切换 ” 快捷键“ 。

7 –> alt+↑/↓

移动一行或多行。

8 –> ctrl+alt+↑/↓

复制选择的一行或多行代码块。

9 –> F3

快速查看源码。

窗口快捷键

1 –> alt+←

全局 后退历史记录。

2 –> alt+→

全局 前进历史记录。

3 –> ctrl+o

快速 Outline 视图：查看当前文件中的类、方法名。

4 –> ctrl+PgDn/PgUp

选项卡之间快速切换。

5 –> ctrl+h

全局搜索对话框。

6 –> ctrl+f

当前文件搜索、替换对话框。

Debug 模式快捷键

1 –> F11

Debug 模式运行程序。

2 –> F5

单步调试进入函数内部。

3 –> F6

单步调试不进入函数内部。

4 –> F8

一直执行到下一个断点。

5 –> F7

由函数内部返回到调用处。

6 –> alt+shift+x

正常运行应用。
